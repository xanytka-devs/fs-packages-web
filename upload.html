Пакеты

Добро пожаловать в официальный сайт по дистрибуции пакетов для Firesteel.

Как опубликовать пакет?

На данный момент единственный способ официально опубликовать пакет - создать Issue в репозитории пакетов.

Как скачать пакет?

Для установки вручную нужно:

  1. Скачать архив с кодом по ссылке:

    https://xanytka.ru/p/{название_пакета}.zip

    * На данный момент все пакеты есть в специальном репозитории.

  2. Разархивировать его в корневую папку движка (не забудьте удалить "xepkg.json").

  3. Собрать проект и радоваться результату.

Для установки через этот сайт нужно:

  1. Скачать необходимый пакет-архив с кодом.

  2. Разархивировать его в корневую папку движка (не забудьте удалить "xepkg.json").

  3. Собрать проект и радоваться результату.
